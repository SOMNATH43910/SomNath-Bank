## 🚀 Project Overview

> **SomNath Bank** is a **production-style full-stack digital banking platform** designed to simulate real-world retail banking workflows with secure authentication, modular architecture, and scalable backend services.

<div align="center">

| 🎯 Feature | 💡 Tech Used | ✅ Status |
|:---|:---|:---:|
| Admin & Customer Portals | React.js + Tailwind CSS | ✅ Done |
| JWT Authentication | Spring Security + JWT | ✅ Done |
| Role-Based Access Control | RBAC (Admin/Staff/Customer) | ✅ Done |
| 40+ RESTful APIs | Spring Boot MVC | ✅ Done |
| Multi-Step Approval Workflows | Service Layer State Machine | ✅ Done |
| Secure Transaction Engine | Debit/Credit + Reconciliation | ✅ Done |
| Real-Time Admin Analytics | Live Metrics Dashboard | ✅ Done |
| Complaint Ticketing System | Lifecycle Tracking | ✅ Done |
| KYC Verification Module | Document Upload + Admin Review | ✅ Done |
| Database ORM | Spring Data JPA + Hibernate | ✅ Done |

</div>

## 🌍 Getting Started

<details>
<summary><b>⚙️ Backend Setup</b></summary>

<br/>

**Prerequisites:**
```
Java 17+   |   MySQL 8.0+   |   Maven 3.8+
```

**Configure `application.properties`:**
```properties
spring.datasource.url=jdbc:mysql://localhost:3306/somnath_bank
spring.datasource.username=your_username
spring.datasource.password=your_password
jwt.secret=your_jwt_secret_key
```

**Build & Run:**
```bash
# Clone the repository
git clone https://github.com/SOMNATH43910/SomNath-Bank.git
cd SomNath-Bank

# Build
mvn clean install

# Run
mvn spring-boot:run
```

Backend runs at → `http://localhost:8080/api`

</details>

<details>
<summary><b>🌐 Frontend Setup</b></summary>

<br/>

```bash
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

Frontend runs at → `http://localhost:5173`

</details>

<details>
<summary><b>☁️ AWS EC2 Deployment</b></summary>

```bash
# 1. Launch EC2 instance (Ubuntu recommended)
# 2. Install dependencies
sudo apt update && sudo apt install openjdk-17-jdk maven git mysql-server -y

# 3. Clone & Build
git clone https://github.com/SOMNATH43910/SomNath-Bank.git
cd SomNath-Bank
mvn clean install

# 4. Run
java -jar target/*.jar

# 5. Open port 8080 in Security Group ✅
```

</details>
